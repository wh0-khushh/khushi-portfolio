# 🚀 Run Locally

Clone the repository

```bash
git clone https://github.com/wh0-khushh/khushi-portfolio.git
```

Navigate to the project

```bash
cd khushi-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
ng serve
```

Visit

```
http://localhost:4200
```

---
